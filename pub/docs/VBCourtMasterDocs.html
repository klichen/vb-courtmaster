Getting Started: VBCourtMasterDocs

  • Start by including the jQuery script in your website 
                        
    <script defer src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.0/jquery.min.js"></script>
  • Then include VBCourtMaster's script 
                        
    <script defer type="text/javascript" src='/src/vb-court-master.js'></script>
  • You are now ready to use the library and create your volleyball court representations, refer to the examples provided in our landing page.

  • A basic set up for a court just used for displaying the players and their information is as follows: 
                        
    const firstExample = document.getElementById('example1')
    const court1 = new CourtGenerator(firstExample)
    court1.createCourt()

    court1.addPlayer(1, "OPP", "Yuji Nishida", "Opposite Hitter", "6'1")
    court1.addPlayer(2, "P2", "Meilin Yuanfei", "Left side Hitter", "6'0 in heels")
    court1.addPlayer(3, "M1", "Haku Lee", "Middle Blocker", "6'3")
    court1.addPlayer(4, "S", "Masahiro Sekita", "Setter", "5'9")
    court1.addPlayer(5, "P1", "Yuki Ishikawa", "Left side Hitter", "6'3")
    court1.addPlayer(6, "M2", "Akihiro Yamauchi", "Middle BLocker", "6'8")

(static) addPlayer(position, tag, name, role, description)

Creates a player representation and adds it on the court at the specified position (only add one player per position)

Parameters:
Name Type Description
position number

A number between 1 and 6 signifying where to add the player (according to volleyball positioning - 1 is right back, 6 is middle back, etc.)
tag string

A tag to display on top of the player representation
name string

Name of the player
role string

Role of the player (Wing spiker, setter, middle blocker, etc.)
description string

Quick description of the player (could be height, weight, spike touch, fun facts, etc.)

(static) createCourt(showRotationBtns, showScore, canDraw, id, initRotation)

Creates a court representation with specified features from parameters

Parameters:
Name Type Description
showRotationBtns boolean

Display the rotate/reset bars at the top, use with functions rotateOnce, resetToCurrent, and resetTransition to allow animations (refer to JS code of the fourth example in our landing website)
showScore boolean

Display the scoreboard feature, allow scorekeeping and tracking stats for each rotation (users can see how many points they earned and lost in each rotation)
canDraw boolean

Display the "Draw" option, allows users to sketch on the court representation
id string

Please provide an id of your choice for the created court
initRotation number

A number between 1 and 6, how to count rotations is up to the developer (typically rotations are counted clock-wise with the first rotation being the one with the setter in position 1)
